How to Get Netscape to open CIDD:

  • Edit Netscape Preferences->Navigator->Applications
  • Select New...
  • Enter 'CIDD Parameter Files' in the Description box
  • Enter 'application/cidd' in the MIMEType box
  • Enter 'CIDD' in the Suffixes Box
  • Select Application check box. Enter '/rap/bin/CIDD -p %s'
  • Click OK-> OK
  • Click on the Links above to Start the Display
